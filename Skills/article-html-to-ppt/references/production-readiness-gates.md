# Production Readiness Gates

Use these gates for non-trivial decks.

## Gate 1: Content Lock

Pass when each slide has a judgment title, major claims are sourced or marked as assumptions, no important source section is silently dropped, and paragraphs are transformed into structure.

## Gate 2: Expression Mode Lock

Pass when every slide has `expression_mode`, `expression_mode_reason`, and a visual component plan when needed. Fail if relationships, flows, systems, ecosystems, causal chains, or spatial metaphors are hidden in prose/cards/tables.

## Gate 3: Architecture Lock

Pass when every slide has one primary anchor, one page archetype, a density label that matches actual content, and a structural reason for high density.

## Gate 4: Spec Priority Lock

Pass when every object has role/priority, low-priority objects are deleted or weakened, and grid/connector/icon/table budgets are explicit.

## Gate 5: Visual Reference Lock

Use for polished, formal, public, client-facing, or style-sensitive decks. Pass when cover and one representative content page are previewed and can be reconstructed as hybrid-editable PPT.

## Gate 6: PPT Implementation Lock

Pass when message-bearing content is editable, raster components are bounded and disclosed, all key text fits, representative slides render nonblank, and object/media counts match expectations.
