## Premium Acceptance Evidence

- Renderer: LibreOffice 26.2.4.2; 9/9 slides rendered successfully.
- QA: `pass`; zero fatal/error/warning issues; editable core and native text ratios both 1.0.
- Rubric: explicit model visual review of this run's nine-slide LibreOffice contact sheet, 15/18, no zero dimensions, `manual_review_required: false`; provenance includes `review_id`, `reviewed_at`, configured model version, and method.
- Evidence binding: case/build/deck IDs, exact PPTX SHA-256, canonical render-report SHA-256, and rendered-slide-set digest all match the accepted artifacts.
- Trust gate: Premium `final` additionally requires real render success, SHA-bound readback/inspection, trusted rubric provenance, score >=14, zero QA errors, and disclosed fallbacks.
- Negative safety: missing/fabricated binding, stale render digest, another-deck replay, and inspection SHA mismatch all fail closed and cannot grant `final`.
- Regression: renderer-unavailable runs remain blocked at verify; renderer-available runs without an explicit rubric stop before delivery.
- Scope boundary: this is LibreOffice Premium acceptance; native Microsoft PowerPoint compatibility remains unverified.
